# Feature Engineering

## Objective

Transform a public credit-risk dataset into a synthetic dataset representing members of a Saudi Jamiya.

---

## Engineered Features

| Feature | Description |
|----------|-------------|
| Monthly Salary | Monthly income of the member |
| Salary Date | Salary deposit day |
| Monthly Contribution | Fixed Jamiya payment |
| On-Time Payments | Successful payments |
| Late Payments | Delayed payments |
| Missed Payments | Missed installments |
| Previous Default | Historical payment issues |
| Trust Score | Reliability score (0–100) |
| Risk Label | Green / Yellow / Red |

---

## Why Feature Engineering?

These features allow the AI Risk Agent to evaluate member reliability while preserving financial privacy.

No real customer data is used.
